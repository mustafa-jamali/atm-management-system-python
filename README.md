ATM Management System (Python)

This is a simple **ATM Management System** built using **Python** and **File Handling**.  
It allows users to check balance, deposit money, and withdraw money.

This project is created for **Python practice** and to understand:
- Functions
- Loops
- Conditional statements
- File handling

---

## Features
- Check account balance
- Deposit money
- Withdraw money
- Balance stored permanently using a text file

---

##  Technologies Used
- Python 3
- File Handling (`.txt` file)

---

## Files Used
- `atm.py` → Main Python program
- `balance.txt` → Stores account balance
